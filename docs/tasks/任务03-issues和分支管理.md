## 创建 issue

1. 进入代码仓库
2. 点击页面中的 “Issue” 标签。
3. 点击 “新建 Issue” 按钮，此时会弹出一个对话框。
4. 在对话框中填写相关信息，包括 Issue 的标题、详细描述，还可根据需要选择指派给的成员、添加标签等。
5. 填写完成后，点击 “提交新 Issue” 按钮，即可完成创建。

   【示例】

- title：创建 login 页面，登录成功后跳转到系统首页

  - 输入的用户名或者密码错误时，显示错误信息
  - 登录成功后跳转到系统首页
  - 页面使用 element-plus 组件来实现
  - UI 主题风格为粉紫色

- 选择责任人
- 选择优先级
- 提交
